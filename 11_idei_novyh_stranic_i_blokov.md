## 11.3 Обязательные новые блоки на главной

1. Блок "Выберите сценарий":
- подобрать станок;
- отправить ТЗ;
- получить ТКП;
- связаться с сервисом.

2. Блок "Коммерческие условия":
- сроки поставки;
- варианты оплаты;
- лизинг;
- гарантия;
- условия ПНР.

3. Блок "Доказательства":
- кейсы;
- цифры внедрений;
- список отраслей;
- клиенты.

4. Блок "Ответы до обращения":
- 6-8 ключевых вопросов закупки и эксплуатации.

## 11.4 Обязательные новые блоки в карточке станка

1. "Для каких задач и деталей подходит"
2. "Ограничения применения" (честно о границах модели)
3. "Таблица ТТХ" (полная, единый формат)
4. "Базовая/опциональная комплектация"
5. "Срок поставки по конфигурациям"
6. "Сервис и гарантия"
7. "Документы и сертификаты"
8. "FAQ по конкретной модели"
9. "Сравнить с 2-3 альтернативами"

## 11.5 Новые блоки для роста глубины просмотра

1. "Связанные решения" с реальными ссылками:
- оснастка;
- роботизация;
- соседние модели.

2. "Следующий логичный шаг":
- после карточки -> сравнение;
- после сравнения -> отправка ТЗ;
- после ТЗ -> консультация инженера.

3. "Материалы по теме":
- статьи;
- видео работы оборудования;
- инструкции по выбору.

## 11.6 Контентные шаблоны для нейропоиска

1. Карточка фактов в начале страницы:
- модель;
- тип;
- ключевые параметры;
- сроки;
- сервис;
- регион.

2. Короткие ответы "вопрос-ответ":
- один вопрос = один четкий факт.

3. Структурированные таблицы:
- без размытой терминологии;
- единые поля у всех моделей.

4. Блок "Почему выбрать эту модель":
- 3-5 проверяемых преимуществ с цифрами.
